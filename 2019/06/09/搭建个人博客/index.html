搭建个人博客
发表于 2019-06-09 | 更新于 2019-06-09
个人博客搭建 | hexo

创建文件夹，命名为myblog

初始化文件夹：

进入myblog文件夹
hexo init ./
或者在myblog文件夹外 hexo init myblog

hexo s 启动本地服务，进行预览

创建一篇新文章

hexo n ‘js运行机制’
创建一篇新文章
source文件夹下 js运行机制.md 文件

tag标签一行一个 前面加‘-’


标题和时间都能改

生成纯静态页面

hexo g生成纯静态页面 .html在public文件夹里

github托管静态网站，实现线上访问

在github上新建一个仓库，命名为
用户名.github.io

修改主页名称

拉到最下边部署
git 仓库名要修改成自己的
deploy:
type:


‘type: ‘的后面一定要有空格

  • 部署后要安装插件 npm install hexo-deployer-git –save
  • hexo d 自动部署到网站上

删除文章：

source文件夹下删掉文章
hexo g 线上重新生成

更换博客的主题

themes


将主题下载到themes目录

可以在github 搜hexo-theme看哪个比较有名
cd themes/
把文件克隆到themes目录下
